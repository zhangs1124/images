# 🎨 AI 圖片生成互動式引導流程

## 📋 觸發方式

當使用者說以下任何一句話時，啟動此引導流程：

- ✅ 「幫我生成一張圖片」
- ✅ 「我想要一張照片」
- ✅ 「生成圖片」
- ✅ 「引導我生圖」
- ✅ 「我需要一張圖」
- ✅ 「做一張圖」

---

## 🎯 引導流程

### 步驟 1：詢問圖片類型

```
🎨 Nano Banana 圖片生成引導

請告訴我，你想要生成什麼類型的圖片？

📸 1. 產品照片（電商、廣告）
🍔 2. 美食照片（菜單、社交媒體）
🏠 3. 空間照片（室內設計、房地產）
👔 4. 時尚照片（服裝、穿搭）
🌆 5. 風景照片（城市、自然）
👤 6. 人物照片（肖像、生活）
🎨 7. 創意圖片（Pop-Art、插畫風格）
📊 8. 資訊圖表（教育、技術說明）
📱 9. 社交媒體（IG、FB 貼文）
🎪 10. 其他（請描述）

請選擇數字或直接描述：
```

### 步驟 2：根據類型詢問細節

#### 2.1 產品照片

```
📸 產品照片 - 讓我了解更多：

1. 什麼產品？
   □ 電子產品（手機、耳機、相機）
   □ 時尚配件（手錶、包包、鞋子）
   □ 家居用品（杯子、擺設、家具）
   □ 美妝產品（香水、化妝品）
   □ 食品包裝
   □ 其他：_______

2. 用途是什麼？
   □ 電商平台（需要去背、白底）
   □ 社交媒體（需要情境、吸睛）
   □ 廣告宣傳（需要創意、高級感）
   □ 產品目錄（需要多角度）

3. 風格偏好？
   □ 簡約專業（純白背景）
   □ 生活情境（放在使用場景中）
   □ 創意藝術（特殊效果）

→ 查看資料庫：prompt.md > 產業專屬 Prompt > 電商
```

- 用途？（菜單 / 社交媒體 / 美食部落格）

### 步驟 5：詢問是否滿意並提供調整選項

```
✅ 圖片已生成並儲存到：d:\project\首尾\【檔名】

你滿意嗎？如果需要調整，可以告訴我：

🎨 色彩調整：
- 「顏色更鮮豔」
- 「色調更溫暖/冷色」
- 「增加對比度」

💡 光線調整：
- 「增加陽光感」
- 「光線更柔和」
- 「改成夜景」

📐 構圖調整：
- 「換個角度」
- 「拉近/拉遠」
- 「改變背景」

✨ 其他調整：
- 「加入【元素】」
- 「移除【元素】」
- 「改變【某個部分】」

或者：
- 「很好，不用調整」
- 「重新生成一張」
- 「用另一個風格」
```

```
使用者說「幫我生成一張圖片」
    ↓
步驟 1：詢問類型（10 個分類）
    ↓

    ↓

    ↓

    ↓

    ↓
步驟 5：生成圖片
    ↓
步驟 6：詢問是否滿意
    ↓
需要調整？
├─ 是 → 根據反饋調整 → 重新生成
└─ 否 → 完成 ✅
```

## 📝 使用範例

### 範例 1：完整引導

```
使用者：「幫我生成一張圖片」
AI：「請選擇類型...（顯示 10 個分類）」
使用者：「美食照片」
AI：「什麼料理？什麼風格？」
使用者：「牛排，專業菜單風格」
AI：「✅ 查看資料庫...提供 3 個選項」
使用者：「選項 A」
AI：「✅ 生成圖片」
AI：「滿意嗎？需要調整嗎？」
使用者：「顏色更鮮豔」
AI：「✅ 重新生成」
```

### 範例 2：快速模式（自動識別）

```
使用者：「生成一張牛排的菜單照片」
AI：「✅ 識別到：美食照片 > 菜單風格」
AI：「查看 prompt.md 餐飲章節...」
AI：「我可以生成【3 個選項】，你要哪一種？」
使用者：「專業菜單風格」
AI：「✅ 生成圖片」
```

### 範例 3：進階需求

```
使用者：「我想要一張 Pop-Art 風格的街頭照片」
AI：「✅ 識別到：進階技巧 > Pop-Art」
AI：「請描述基礎照片：人物、服裝、場景」
使用者：「女孩、休閒服、街頭」
AI：「✅ 查看資料庫...」
AI：「生成：真實照片 + Pop-Art 插畫疊加」
```

## ⚠️ 重要提醒

1. **每次都要查看資料庫**
   - 不要憑記憶或即興創作
   - 使用 view_file 或 grep_search 查看 prompt.md

2. **提供選擇而非直接生成**
   - 讓使用者選擇風格
   - 確保符合需求

3. **記錄 Prompt 來源**
   - 告訴使用者使用了哪個 Prompt
   - 標註來自 prompt.md 的哪個章節

4. **持續優化**
   - 生成後詢問是否滿意
   - 根據反饋調整
   - 直到使用者滿意
