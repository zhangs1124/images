# 🎨 AI 圖片 Prompt 優化器

## 📋 使用方法

### 步驟 1：輸入簡單描述
例如：「一隻小狗在湖邊跑」

### 步驟 2：選擇風格
從以下風格中選擇一個或多個：
1. 寫實攝影風格
2. 吉卜力動畫風格
3. 3D 卡通風格（皮克斯）
4. 水彩插畫風格
5. 極簡線條風格
6. 賽博朋克風格
7. 復古海報風格
8. 油畫藝術風格

### 步驟 3：獲得優化後的 Prompt
系統會自動添加：
- 詳細的視覺描述
- 光線和色彩設定
- 構圖和視角
- 材質和質感
- 氛圍和情緒
- 技術參數

## 🎨 常用元素庫

### 光線類型
- 金色時刻（Golden Hour）：日出/日落時的溫暖光線
- 藍調時刻（Blue Hour）：黎明/黃昏的藍色光線
- 柔光（Soft Light）：陰天或室內的柔和光線
- 側光（Side Light）：從側面照射，強調立體感
- 逆光（Backlight）：從背後照射，營造輪廓光
- 戲劇性光（Dramatic Light）：強烈對比的光影

### 色調風格
- 溫暖色調：橙、黃、紅為主
- 冷色調：藍、綠、紫為主
- 中性色調：灰、白、黑為主
- 高對比：明暗對比強烈
- 低對比：柔和過渡
- 粉彩色：淡雅柔和的色彩

### 構圖方式
- 三分法：將畫面分為九宮格
- 中心構圖：主體位於畫面中央
- 對角線構圖：沿對角線排列元素
- 框架構圖：用前景元素框住主體
- 引導線構圖：用線條引導視線
- 對稱構圖：左右或上下對稱

### 氛圍關鍵詞
- 歡快：joyful, cheerful, happy
- 寧靜：peaceful, serene, calm
- 溫馨：warm, cozy, heartwarming
- 夢幻：dreamy, ethereal, magical
- 神秘：mysterious, enigmatic
- 戲劇性：dramatic, intense
- 浪漫：romantic, tender
- 活潑：lively, energetic, vibrant

## 📝 快速優化流程

### 輸入分析
1. 識別主體（人物/動物/物品）
2. 識別動作（靜態/動態）
3. 識別場景（室內/室外/特定地點）
4. 識別時間（白天/夜晚/特定時刻）

### 自動添加
1. 光線描述（根據時間和場景）
2. 色彩描述（根據氛圍）
3. 細節描述（材質、質感）
4. 氛圍描述（情緒、感覺）
5. 技術參數（解析度、風格）

### 輸出格式
- 中文 Prompt（完整描述）
- English Prompt（國際通用）
- 關鍵參數（解析度、風格標籤）

## 💡 產業優化技巧

### 1. 識別產業需求
- 電商：強調產品細節、去背、多角度
- 餐飲：強調食物質感、色彩、誘人感
- 房地產：強調空間感、光線、舒適度
- 時尚：強調質感、風格、時尚感
